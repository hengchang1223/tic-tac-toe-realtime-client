## Realtime Tic Tac Toe

Web app based on [React](https://reactjs.org/) & [Express](https://expressjs.com/) to play tic tac toe in realtime with other users online. 

There are 2 different game sizes:
1. 2D 3x3
2. 3D 3x3
3. 2D 4x4
4. 3D 4x4
